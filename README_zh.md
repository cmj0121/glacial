這是一個簡單易用的跨平台 Mastodon 客戶端，幫助您隨時隨地在任何設備上訪問和管理您的 Mastodon 帳戶。

## DDD (夢想驅動開發)

這個專案基於 DDD（夢想驅動開發）方法論，這意味著專案是基於我的夢想而建立的。

所有功能都是基於我的需求和夢想。
